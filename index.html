Welcome to metvurst!

metvurst has two components:

METeorological Visualisation Utilities using R
Using R for Science and Teaching

For examples and showcases of the metvurst package see the metvurst blog

metvurst is implemented in R:
In case you want to install metvurst, make sure you have devtools installed and loaded and then simply type:

install_github('metvurst', 'tim-salabim')
library(metvurst)

Note:
this package is only a skeleton a the moment. I will progressively add bits'n'pieces (especially documentation and examples) in the near future.
